    Writing Bundles

    Bundles are a way to provide multiple plugins in a single package. This is useful when you have a set of plugins that are related to each other or want to provide a set of Plugins that represent a larger install. For example, you may have a set of plugins that all install software that is used for video editing or software development. You can then offer it as a single bundle that can be installed with a single run instead of multiple.

    • Read Getting Started

    Creating a Juno Pipeline Bundle

    Juno Innovations provides a default pipeline that it uses for its own projects. This pipeline is offered as a bundle for others to install and use. We will walk through how this is created.

    Calling Other Plugins

    Because the Juno Pipeline is a bundle, it will call other plugins to install the software it needs. This is done by calling the plugins() function that is provided by the terra loaders. This allows Juno to load the Blender or Nuke plugins and install them as needed.

    This can then be chained together and all versions of the software can be locked by Juno for better support.
